原文：

大家好，我一直很喜欢我的RPi 4路由器，这似乎是一个热门话题，所以我想写一篇简短的文章来帮助人们开始自己的设置。如果您有任何反馈或想提出编辑建议，请随时在GitHub上打开问题或PR 433。

所以你想建立一个树莓派路由器

随着千兆速度家庭互联网连接在美国和其他地方的日益普及，需要比许多“一体式”设备性能更高的路由硬件。对于希望利用SQM等高级流量整形技术的用户来说尤其如此，SQM是众所周知的CPU密集型技术。树莓派4单板计算机价格实惠，无处不在，并围绕其价格点附近的任何消费设备运行。所有这些品质使其成为成为您家庭网络下一个基础的理想人选。

它有什么好处

  • 禁用SQM的全千兆位路由性能。CPU当然能有更多能力，只受网络接口速度的限制。
  • ~800 Mbps上/下，启用SQM。
  • 如果您愿意，可以提供大量额外的RAM和处理能力来运行其他服务。
  • 在没有移动部件、低功耗和非常紧凑的占地面积下以极具竞争力的价格完成上述所有工作。

它对什么不太好

  • 维修您的无线设备。与体面的一体式消费者路由器相比，Pi的板载WiFi实现真的很可怜。相反，计划用另一台设备来补充Pi，作为无线接入点。

你需要什么

  • 树莓派4板。对于这个应用程序来说，1GB的RAM就足够了，但更多不会有什么坏处。
  • 电源或PoE帽子。
  • 任何常见尺寸的优质microSD卡；与普通塑料消费网络路由器相比，即使是8GB也是巨大的空间。
  • 专为被动冷却而设计的外壳。以下是您可能会考虑的几个选项：
    • 弗利尔克 527
    • 氩40一 349
    • Argon40 Neo 372
    • 极客蠕虫盔甲 429
  • USB到以太网接口。任何基于RTL8153芯片组的加密狗都可以。TheTP-Link UE300 1.6k是一个受欢迎的选择，效果很好。

可以在所谓的“杆上路由器”配置中使用Pi，只需使用单个板载以太网端口和VLAN。此配置需要使用托管交换机，超出了本指南的范围。

示例构建和总成本

  • 树莓派4 2GB：35美元
  • 内陆USB-C电源：6美元
  • Micro Center 64GB microSD卡：7美元
  • Argon Neo 案例：15美元
  • TP-Link UE300：14美元

总计：77美元

请注意，由于持续的全球不愉快，2GB Pi 4最近价格上涨至45美元。据说这是一个临时措施，但目前我觉得4GB版本的价值更高，只有55美元。选择4GB主板将使总构建成本达到97美元。

入门指南

组装好您的Pi后，是时候将OpenWrt图像写入您的microSD卡了。一种选择是使用wulfy23的出色构建 2.3k这应该让你以最小的麻烦来启动和运行。另一个是使用OpenWrt提供的库存图像；然而，库存图像缺少USB加密狗的驱动程序。最后，也是我们将在这里探索的，是用所需的软件包构建自定义图像。幸运的是，OpenWrt固件选择器网站可以轻松请求这样的图像。

要开始，请导航到固件选择器服务器https://firmware-selector.openwrt.org/3.1k。开始在搜索字段中键入Raspberry Pi 4，然后选择Raspberry Pi 4B/400/4CM (64bit)现在扩展Customize手风琴。

我们现在可以自由地将其他软件包添加到我们的固件映像中。很好！继续将这三个软件包添加到列表末尾：kmod-usb-net-rtl8152 luci luci-ssl。这将确保USB以太网驱动程序和LuCI Web界面都融入我们的自定义图像中。

单击“Request Build”，并在等待图像构建时为咖啡充值。准备就绪后，下载FACTORY (EXT4)图像。解压缩压缩图像，最后使用您选择的工具将其闪存到您的microSD卡上。

上网

将Pi的板载以太网端口连接到局域网。将您的广域网连接连接到USB加密狗，但尚未将其连接到Pi。在浏览器中导航到http://192.168.1.1，无密码登录。立即执行以下操作：

  • 导航到System -> Administration -> Router Password并配置strong密码。
  • 导航到System -> Administration -> SSH Access，并将SSH服务器配置为仅监听LAN接口。
  • 如果需要，请为SSH登录提供公钥System -> Administration -> SSH-Keys

现在LuCI（OpenWrt基于Web的配置接口）和SSH都已锁定，您可以连接USB以太网加密狗。导航到Network -> Interfaces -> Devices，并确认您看到一个新的eth1设备。如果没有，请从System -> Reboot重新启动路由器。

单击返回Interfaces选项卡。使用正确的互联网服务协议添加一个名为WAN的新接口，并将其附加到新的eth1接口。根据您选择的协议完成任何其他配置步骤，然后保存并应用您的更改。

如果您的ISP要求您的WAN设备通过特定的VLAN进行通信，请使用“自定义”设备字段以DEVICE.VLANID的形式提供VLAN ID，例如eth1.10。

导航到Firewall Settings，并确认新的广域网接口已设置为防火墙区域。

在此步骤中，您还需要导航到“防火墙设置”选项卡，并将“WAN”分配为防火墙区域。

你现在应该连接到互联网了！

最大限度地提高性能

在当前配置中，您的路由器将毫不费力地路由对称千兆位连接。然而，我们可以做几件事来从Pi中捻来获得更多的性能。

配置调整

  • 导航到Network -> Interfaces -> Global network options并启用Packet Steering。此功能有助于确保属于特定流的所有数据包都由相同的处理核心处理，从而提高性能。看到这里 152对该概念进行有用的介绍。
  • 导航到Network -> Firewall，并验证Software flow offloading设置。如果您打算使用SQM，您需要禁用此功能。如果您不会使用SQM，请立即启用它。

启用irqbalance

IRQ是网络接口通知操作系统有工作要做的机制。默认情况下，这些请求将由单个CPU内核处理，这在单个内核无法跟上的情况下可能导致瓶颈。

这就是irqbalance软件包的用处。此守护程序监控处理器负载和IRQ卷，并在不同的CPU内核上分配IRQ，缓解瓶颈。

使用irqbalance非常简单：只需从System -> Software安装，然后重新启动路由器或从System -> Startup手动启动服务，以确保守护进程正在运行。您现在应该在Status -> Processes中看到anirqbalance进程。

超频CPU

只需简单更改配置，您的新路由器的CPU就可以超频到2 GHz。算一算——这是所有核心免费33%的性能提升！虽然更高的频率必然会转化为更多的热量，但为被动冷却或任何主动冷却解决方案设计的优质外壳应该不会让您的Pi保持在舒适的温度。

虽然甚至更高的超频是可能的（尽管有传统智慧，但您的保修不再失效 42截至Pi 4），我认为对大多数人来说，多余的果汁不太可能值得挤压。

要启用超频，只需将以下行添加到/boot/config.txt的底部，然后重新启动：

over_voltage=6
arm_freq=2000
